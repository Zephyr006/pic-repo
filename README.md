PicGo 配置注意事项
- 存储路径：指的是图片上传后在远程仓库中的存储路径，不能以“/”开头，一般以“/”结尾，作为远程仓库中的文件夹路径（不能是绝对路径）
- 自定义域名，使用：https://cdn.jsdelivr.net/gh/GitHub用户名/仓库名
